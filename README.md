# Spring Boot JWT API

REST API built with Spring Boot 3 and Java 17 focused on secure authentication using JWT and stateless configuration.

---

## 🚀 Features

- JWT Authentication
- Stateless Spring Security configuration
- Layered architecture (Controller, Service, Repository)
- Exception handling
- Relational database integration

---

## 🛠 Tech Stack

- Java 17
- Spring Boot 3
- Spring Security
- JWT
- Maven
- JPA / Hibernate

---

## ▶️ Run the project

Clone the repository:

```bash
git clone https://github.com/Roronoa179/springboot-jwt-api.git
```

Enter the project folder:

```bash
cd springboot-jwt-api
```

Run the application:

```bash
mvn spring-boot:run
```

---

## 📌 Purpose of this project

This project was built as backend practice to strengthen:

- Spring Security
- Authentication flows
- HTTP status handling (401 vs 403)
- Clean backend architecture
